# ❌⭕ Python Tic Tac Toe

A simple Command Line Interface (CLI) Tic Tac Toe game built using Python.
This project was created to practice basic programming logic (Lists, Loops, and Conditionals).

## How to Play
1. Run the program.
2. Player 1 (X) and Player 2 (O) take turns choosing a grid position (1-9).
3. The first player to align 3 symbols (horizontally, vertically, or diagonally) wins the game.

## How to Run
Make sure Python is installed on your computer, then run this command in your terminal:

```bash
python game.py
